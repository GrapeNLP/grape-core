Detailed Description

template<typename TagInput, typename TagOutput>
class grapenlp::ns_rtno< TagInput, TagOutput >

Recursive transition network with output (RTNO) and numbered states, that is, a "number" field in each state to be used for printing the state labels (e.g. q0, q1, etc.). These RTNOs include a state counter to automatically assign unique numbers to the states as they are added to the RTNO.

Template Parameters
TagInputthe type of the input tags (e.g. for letter RTNOs it would be unichar)
TagOutputthe type of the output tags (e.g. for letter RTNOs it would be unichar)
