Detailed Description

template<typename TagInput, typename TagOutput>
class grapenlp::ns_rtno< TagInput, TagOutput >::state

State of the recursive transition network with output and numbered states; this class simply extends the RTNO states with a "number" field and methods for generating the state label (e.g. q0, q1, etc.)
