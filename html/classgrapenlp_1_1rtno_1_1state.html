Detailed Description

template<typename TagInput, typename TagOutput>
class grapenlp::rtno< TagInput, TagOutput >::state

State of the recursive transition network with output; a RTNO state comprises a flag indicating whether the state is final or not, and sets of outgoing transitions towards other states of the RTNO. Ingoing transitions are not needed since the the parsing algorithms just need to traverse the RTNO from the initial state to the final ones.
